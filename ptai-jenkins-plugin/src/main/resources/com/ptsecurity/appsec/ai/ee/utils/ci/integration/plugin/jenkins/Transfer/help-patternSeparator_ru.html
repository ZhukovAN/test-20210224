Регулярное выражение, используемое в качестве разделителя в перечнях шаблонов анализируемых и игнорируемых файлов.

Указанные перечни поддерживают указание множественных шаблонов, разделение которых по умолчанию осуществляется с помощью регулярного выражения 

[, ]+

Однако, используемое по умолчанию регулярное выражение делает невозможным использование шаблонов имен файлов или каталогов, содержащих пробелы. Переопределение этого параметра позволяет решить эту проблему
