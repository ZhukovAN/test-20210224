PT AI vulnerability analysis mode. project name. Possible options:

  • Asynchronous mode
    Build pipeline starts application secutity testing (AST) job and continues its execution. The only result file that is published to CI workspace folder is result.url that contains PT AI's REST API endpoint to get scan results
  • Synchronous mode
    Build pipeline starts application secutity testing job and waits its completion. During last stage of AST job PT AI generates reports and makes security policy assessment (if any). This work mode allows you to define if CI build step should be marked as failed if policy assessment is failed and to define reports to be generated on scan complete
