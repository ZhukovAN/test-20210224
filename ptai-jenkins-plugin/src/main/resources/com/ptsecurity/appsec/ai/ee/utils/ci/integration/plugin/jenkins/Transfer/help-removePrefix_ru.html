Префикс полного пути к файлу, который не должен быть использован при передаче файла.

В ходе работы структура каталогов формируется относительно базового каталога, который, как правило, является рабочей папкой Jenkins.
Зачастую, для анализа кода нет необходимости в передаче полной структуры каталогов. Например, если файлы размещаются в каталоге target/deployment/images/**/, для упрощения структуры каталогов можно убрать префикс target/deployment Это позволит при анализе создать каталог images непосредственно в корневом каталоге, а не в target/deployment
При формировании этого пути могут быть использованы переменные окружения Jenkins.

При использовании этого параметра все пути к анализируемым файлам должны начинаться с него.
